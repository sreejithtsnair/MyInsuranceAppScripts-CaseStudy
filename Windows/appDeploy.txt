Windows –
pip install -r requirements.txt
py project/init/init_db.py
py runserver.py
Launch the host URL – http://localhost:5000/

Linux –
Create a script using: vi scr
Write the commands in the script: pip3 install -r requirements.txt 
                        python3 project/init/init_db.py
                        python3 reserver.py
To execute the script: ./ scr
Launch the host URL – http://localhost:5000/
